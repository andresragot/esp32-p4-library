\doxysubsection{Detailed Description}
Represents a camera in a 3D space, managing its properties and transformations. 

The \doxylink{class_ragot_1_1_camera}{Camera} class provides functionality to set and get camera properties such as field of view, near and far planes, aspect ratio, and target position. It also computes projection, view, and combined view-\/projection matrices, and provides methods for projecting world coordinates to normalized device coordinates (NDC).
