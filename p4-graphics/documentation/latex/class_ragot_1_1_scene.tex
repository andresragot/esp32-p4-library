\doxysubsection{Detailed Description}
Class for managing a 3D scene. 

The \doxylink{class_ragot_1_1_scene}{Scene} class provides methods to manage nodes, cameras, and scene traversal. It allows adding and removing nodes, setting the main camera, and traversing the scene graph.
