\doxysubsection{Detailed Description}
Driver for the ST7789 LCD panel. 

This class provides methods to initialize, deinitialize, and send frame buffers to the ST7789 LCD panel. It inherits from the \doxylink{class_ragot_1_1_driver_l_c_d}{Driver\+LCD} class and implements the necessary methods for LCD operations.
