\doxysubsection{Detailed Description}
Represents a node in a scene graph for 3D rendering. 

The \doxylink{class_ragot_1_1_node}{Node} class extends the \doxylink{class_ragot_1_1_transform}{Transform} class to include child nodes, allowing for hierarchical transformations and management of child nodes.
