\doxysubsection{Detailed Description}
Base class for components in the \doxylink{namespace_ragot}{Ragot} engine. 

The \doxylink{class_ragot_1_1_component}{Component} class serves as a base class for all components in the \doxylink{namespace_ragot}{Ragot} engine. It allows for the management of a collection of components, providing methods to add and remove components, and access the list of components.
