\doxysubsection{Detailed Description}
Base class for LCD drivers. 

This class provides an interface for LCD drivers, including methods for initialization, deinitialization, setting pixels, and sending frame buffers. It also provides access to the LCD panel\textquotesingle{}s width, height, pixel format, and handler.
