\doxysubsection{Detailed Description}
This file defines the Driver\+LCD class, which serves as a base class for LCD drivers in the \doxylink{namespace_ragot}{Ragot} engine. 

\begin{DoxyAuthor}{Author}
Andrés \doxylink{namespace_ragot}{Ragot} (github.\+com/andresragot)
\end{DoxyAuthor}
The Driver\+LCD class provides an interface for initializing, deinitializing, setting pixels, and sending frame buffers to an LCD panel. It includes methods to get the width, height, pixel format, and handler of the LCD panel. The class is designed to be inherited by specific LCD driver implementations, such as the EK79007 driver. \begin{DoxyVersion}{Version}
