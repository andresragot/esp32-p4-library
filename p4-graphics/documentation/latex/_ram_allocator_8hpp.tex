\doxysubsection{Detailed Description}
This file implements a custom memory allocator for PSRAM in the \doxylink{namespace_ragot}{Ragot} engine. 

\begin{DoxyAuthor}{Author}
Andrés \doxylink{namespace_ragot}{Ragot} (github.\+com/andresragot)
\end{DoxyAuthor}
The PSRAMAllocator class provides a way to allocate and deallocate memory in PSRAM with specific flags. It is designed to be used with standard containers like std\+::vector, allowing for efficient memory management in embedded systems.
