\doxysubsection{Detailed Description}
This file implements the Logger class, which provides a singleton logger for the \doxylink{namespace_ragot}{Ragot} engine. 

\begin{DoxyAuthor}{Author}
Andrés \doxylink{namespace_ragot}{Ragot} (github.\+com/andresragot)
\end{DoxyAuthor}
The Logger class allows logging messages with different severity levels (INFO, WARNING, ERROR). It supports formatted logging using printf-\/style format strings and can be used across different platforms. The logger can be configured to set the log level, and it uses the ESP-\/\+IDF logging system on ESP platforms. \begin{DoxyVersion}{Version}
