\doxysubsection{Detailed Description}
Manages the asset paths for the application. 

The \doxylink{class_ragot_1_1_assets}{Assets} class provides a singleton instance to manage asset paths, allowing for easy retrieval of asset files. It initializes the base path based on the executable file path, and provides a method to get the full path of an asset by its name.
