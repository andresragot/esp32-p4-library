\doxysubsection{Detailed Description}
Class for generating revolution meshes. 

This class generates a mesh by revolving a 2D profile around an axis. It inherits from the \doxylink{class_ragot_1_1_mesh}{Mesh} class and implements the methods to generate vertices and faces.
