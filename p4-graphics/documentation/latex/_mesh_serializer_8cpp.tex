\doxysubsection{Detailed Description}
This file implements the Mesh\+Serializer class, which provides methods to serialize a Mesh object to an OBJ file format. 

\begin{DoxyAuthor}{Author}
Andrés \doxylink{namespace_ragot}{Ragot} (github.\+com/andresragot)
\end{DoxyAuthor}
The Mesh\+Serializer class allows saving a Mesh object to an OBJ file, which is a common format for 3D models. It handles the serialization of vertices and faces, ensuring that the data is written in a format compatible with OBJ files. The class is designed to be used in graphics applications where exporting 3D models is required. \begin{DoxyVersion}{Version}
